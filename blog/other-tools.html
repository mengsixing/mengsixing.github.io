# Mac 远程登录 Windows

今天遇到个问题，需要远程到服务器端处理，可是服务器居然是 windows 的（很 low 的感觉），不能直接用 ssh 命令远程连接，最后找到了 Microsoft Remote Desktop for Mac 工具。

这个工具官方是推荐直接在 appStore 下载，可是在中国区的 appStore 里却找不到，这里使用的是 beta 版，直接下载就能用。
